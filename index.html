Acerca de HTML5

Publicado 14 Agosto 2019

Lo primero que hay que entender es que es el HTML.Por sus siglas al ingles, es HYPER TEXT MARKUP LANGUAGE .Donde viendolo mejor su nombre nos indica mucho y poco para los que somos novatos, de manera que sin caer en tecnicismos transcendentales,podemos definirlo como un lenguaje hecho para la estructuracion de contenido en un documento *.html los cuales son la base en la organizacion de contenido en paginas web.

Ahora enfocandonos en los fundamentos de este lenguaje, lo mas elemental son las denominadas etiquetas representados por <nombre de etiqueta> siendo estos los elementos semanticos propios del lenguaje que tienen diversas funciones como las de estructurar el texto, cargar y determinar archivos multimedias como imagenes o videos, enlazar otros documentos, enlazar otros sitios web, entre muchas otras funciones.Por los momentos tengo conocimiento de dos tipos de etiquetas, la cerradas las cuales son utilizadas con dos etiquetas, una de apertura y otra de cierre(</etiqueta>) y que generalmente abrazan un texto al cual le aplican una accion de estructura, estilo, vinculacion o cualquier otra cosa posible en el mundo de la pg webs.La segunda version de las etiquetas son las abiertas las cuales solo usan la etiqueta de apertura y no tienen accion directa sobre un texto.Otro concepto de este lenguaje destacable para esta sección es el atributo el cual es un complemento de la etiqueta representado de la siguiente manera < tag atributo="valor de atributo> y en algunos casos es imperativo su uso para la correcta accion de una etiqueta particular.

Relacion entre HTML & CSS

Publicado 14 Agosto 2019

En esta entrada hablaremos de la relacion del CSS y HTML. Resulta que los estandares actuales para la creacion de pg webs estipulan el uso de un lenguaje CASCADING STYLE SHEET, el cual sustituye y amplia las funciones de "estilar" el contenido del documento HTML. Este dispone de su propia sintaxis donde se habla de selectores, reglas de estilo y propiedades, con este lenguaje modifica la fuente, el tamaño de fuente, la estetica de fondo, los margenes entre otras acciones, que ya mas adelante sobre la marcha de mi aprendizaje ire actualizando en este post.

Algunas propiedades de css
Propíedad CSS Descripción Valores""
font-family selecciona una fuente monospace, san-serif, serif, cursive etc
font-size determina el tamaño de la fuente px, em, %, etc
font-weight determina el espesor de letra bold, bolder, inherit etc
font-style determina el estilo inclinado o no del texto italic, monospace, inherit, normal, etc
padding modifica el tamaño del area del elemento px, em, % etc
margin ajusta el margen de todo apartir de un elemento referencia en sentido diagonal px
margin-top ajusta el margen de todo apartir de un elemento referencia alejandose del tope px
margin-bottom ajusta el margen de todo apartir de un elemento referencia alejandose del fondo px
margin-left ajusta el margen de todo apartir de un elemento referencia alejandose del borde izquierdo px
margin-right ajusta el margen de todo apartir de un elemento referencia alejandose del borde derecho px
line-height ajusta la separacion entre lineas de texto px, em, % etc
text align ajusta la posicion del texto normal, center, rigth, end, left etc
